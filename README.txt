******************************************************************************
   	  APPLICATION POUR LA GESTION DES ATHLETES ET EPREUVES DE NATATION
				  POUR LES JEUX OLYMPIQUES
******************************************************************************


-Cet application possède 3 Interfaces :
	-> Spectateur
	-> Athlète
	-> Admin

L'interface Spectateur ne requiert aucune connexion au préalable.
Il est possible depuis cet interface de visualiser les epreuves à venir, 
le classement des pays ainsi que les resultats d'une épreuve donnée.

L'interface administrateur permet de gérer les différents athlètes, épreuves à venir, 
et les participations aux épreuves.
A la premiere ouverture de l'application aucun athlète n'est enregistré.
Il faut donc crée des athlètes, et des épreuves ainsi que des particiaptions. 
(les particiaptions correspond à l'inscription d'un athlète à une épreuve).

Une fois le compte créer, il est possible d'acceder à l'interface de l'athlète. 
L'athlète à la possibilité de visulaiser: ses informations, les épreuves à venir, et les médailles obtenues.
